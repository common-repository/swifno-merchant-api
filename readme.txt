This plugin enables your website look for courier services ready to make delivery from one location to another.

== Description ==

The Swifno shipping provides for rating by the customers/users. The app is built in such a way that it enables customers and users alike, to rate their chosen courier service provider after delivery is made. This in turn helps us at Swifno to evaluate the courier service providers registered on our platform.



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Make sure you have the plugin 'WooCommerce' installed on your wordpress development
2. Upload `swifno.php` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress an extra menu `Swifno Shipping Merchant API` would apprear amonst the menu in your dashboard of your admin panel
4. Go through the `Swifno Shipping Merchant API` and select 'Set Merchant Key'.  and set api mode to be either in Live or Test.
5. Go through the `Swifno Shipping Merchant API` once more and select 'Swifno Settings' and fill the required field, Package Group, Package Category, Package Size, Package Delivery Speed, Pickup time from, Pickup time to, Insurance & Item-value.
